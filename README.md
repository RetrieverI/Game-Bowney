Game - Kytt

A random top-down 2d survival game where you fight off waves of enemies using your bow, melee and other tools at your disposal.


## Controls
- Move: W / A / S / D
- Dash: Space
- Bow: Left click
- Triple bow: Right click
- Melee: F
- Heal: Q
- Trap: E
- Pause: Esc
